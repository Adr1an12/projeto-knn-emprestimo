# Modelo de Classificação KNN

Projeto desenvolvido para prever aprovação de empréstimos utilizando o algoritmo KNN (K-Nearest Neighbors).

## Tecnologias utilizadas

- Python
- Pandas
- Scikit-Learn

## Etapas realizadas

- Leitura do dataset
- Análise exploratória
- Normalização dos dados
- Treinamento do modelo KNN
- Avaliação do modelo
- Teste manual de previsão
